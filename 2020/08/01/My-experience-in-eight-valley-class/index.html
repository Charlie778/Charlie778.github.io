My experience in eight valley class

Nowadays,i attend a project called Eight Valley which is a project about the new retail .This a project which contains a roadshow to show the strangers what we learn during the summer vacation.So far,I have taken part in three online courses,I have learn some key points about the profit and some characteristics of the new retail.Before that,the teacher let us to search for the history of the Squatter communities by ourselves ,this can help us find the earliest Squatter communities,and make the compare between the common retail and the new retail easier.After the self-research,i found that it needs some new technologies for common retail to develop to the new retail.These technologies contains special vending machines and these vending machines should have some uncommon features like no labor is needed for this machine to work.Besides this, the teacher also divided us into small departments like data analyst,science and technology personnel,interaction designer and the product manager.We have different duties and responsibilities.I feel good after taking these courses.
